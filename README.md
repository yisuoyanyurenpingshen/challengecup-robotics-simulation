# 挑战杯机器人仿真项目记录

本仓库用于记录本人围绕挑战杯机器人仿真项目的长期学习、实验和工程推进过程。

## 项目背景

本项目主要围绕机器人仿真、GPU 实验、视觉模型测试、ROS2/仿真学习以及挑战杯项目材料积累展开。

当前使用远程服务器进行实验，服务器配置包括：

- Ubuntu 20.04
- Intel i7-12700K
- NVIDIA RTX 3090 Ti 24GB

本仓库的重点不是保存所有大文件，而是保存项目过程中的关键代码、实验记录、技术笔记、配置说明和阶段性结果。

## 使用原则

- 代码、文档、日志、配置文件可以上传到 GitHub。
- 大型训练集、模型权重、视频、压缩包、conda 环境不直接上传到 GitHub。
- 服务器 IP、账号、密码、SSH 密钥、Token 等敏感信息禁止上传。
- 每次重要修改都通过 Git commit 保存，形成可回溯的项目历史。

## 仓库结构

```text
docs/       技术笔记、方案分析、学习总结
logs/       每日工程日志
scripts/    实验脚本、测试脚本、工具脚本
configs/    环境配置、实验配置说明
results/    轻量级实验结果、表格、截图说明
datasets/   数据集说明，不存放大型原始数据集
weights/    模型权重说明，不存放大型权重文件
envs/       环境说明，不存放完整虚拟环境
